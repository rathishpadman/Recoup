# Recoup Query Answering

## When To Use

Use when a query response must explain a Recoup case, action, record, audit event, or open dependency.

## Procedure

1. Identify the relevant case, line, customer, action, or audit scope.
2. Retrieve cited records through approved read tools.
3. Answer with record IDs and deterministic basis.
4. State when live model execution or connector access is disabled.

## Validation

- Do not answer without cited records.
- Do not approve, reject, or dispatch an action.
- Do not calculate dollar amounts.
- Do not write to ERP.
