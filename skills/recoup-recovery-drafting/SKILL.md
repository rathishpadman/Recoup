# Recoup Recovery Drafting

## When To Use

Use when an invalid or partial deduction decision needs a draft recovery packet or customer-facing outreach.

## Procedure

1. Start from an existing decision ID.
2. Confirm the decision includes evidence documents and record IDs.
3. Use only the action tool output for proposed amounts.
4. Label every artifact as pending human approval.

## Validation

- Do not calculate dollar amounts.
- Do not exceed the deterministic decision delta.
- Do not send customer correspondence.
- Do not write to ERP.
